# 🐱 DinoDash - 안드로이드 횡스크롤 러너 게임

구글 크롬 공룡 게임에서 영감을 받아 만든 간단한 안드로이드 러너 게임입니다.
귀여운 삼색 고양이가 주인공입니다!

## 📱 게임 소개

**DinoDash**는 Kotlin으로 개발된 네이티브 안드로이드 게임입니다. 귀여운 삼색 고양이가 자동으로 달리면서 다가오는 물웅덩이를 점프로 피하는 간단하지만 중독성 있는 게임입니다!

## ✨ 주요 기능

### 🎮 게임플레이
- **간단한 조작**: 화면을 탭하면 점프!
- **무한 러너**: 끝없이 계속되는 게임플레이
- **점수 시스템**: 장애물을 피할 때마다 점수 획득
- **최고 점수 저장**: 최고 기록이 자동으로 저장됨
- **쉬운 재시작**: 게임 오버 후 바로 재시작 가능

### 🐱 캐릭터 & 스킨
- **삼색 고양이**: 흰색 바탕에 검은 점과 주황색 점무늬 (기본)
- 🌟 **골드 스킨**: 점프 1000번 달성 시 황금 고양이로 변신!
- 💀 **해골 스킨**: 비밀 코드 `ddonddo` 입력 시 해골 고양이!
- **픽셀 아트 스타일**: 귀여운 16x16 픽셀 아트

### 🏆 업적 시스템 (20개)
- **점프 업적**: 첫 점프, 100회, 500회, 1000회
- **점수 업적**: 10점, 50점, 100점, 200점, 500점
- **플레이 업적**: 10회, 50회, 100회 플레이
- **죽음 업적**: 첫 죽음, 10번, 50번, 100번 죽기
- **시간 업적**: 밤/아침 시간대 플레이
- **특별 업적**: 완벽주의자, 스피드 러너
- **업적창 스크롤**: 모든 업적을 스크롤해서 확인 가능

### 🎨 그래픽 & 효과
- 💧 **물웅덩이 장애물**: 고양이가 싫어하는 다양한 크기의 물웅덩이
- **반짝이는 효과**: 물웅덩이가 주기적으로 반짝임
- 🏔️ **배경 요소**: 구름과 초록색 산이 여러 레이어로 스크롤
- **부드러운 애니메이션**: 달리는 다리, 흔들리는 꼬리

### 🔊 사운드 효과
- **점프 소리**: 점프할 때마다 재생
- **게임 오버 소리**: 물웅덩이에 닿았을 때 재생

## 📥 다운로드

최신 버전의 APK는 [GitHub Releases](https://github.com/hanjungwoo3/dinodash/releases)에서 다운로드할 수 있습니다!

### 설치 방법
1. 위 링크에서 최신 APK 다운로드
2. 안드로이드 기기에서 "알 수 없는 출처" 앱 설치 허용
3. APK 파일 실행하여 설치
4. 게임 즐기기! 🎮

## 🎯 게임 방법

1. **시작**: 게임을 실행하면 "탭해서 시작하기" 메시지가 나타납니다
2. **업적**: 🏆 버튼을 눌러 20개의 업적 확인 가능
3. **비밀 코드**: 💀 버튼을 눌러 `ddonddo` 입력 시 해골 스킨 획득!
4. **점프**: 화면 아무 곳이나 탭하면 귀여운 고양이가 점프합니다
5. **피하기**: 다가오는 물웅덩이를 점프로 피하세요 (고양이는 물을 싫어해요!)
6. **점수**: 물웅덩이를 성공적으로 피할 때마다 점수가 올라갑니다
7. **게임 오버**: 물웅덩이를 밟으면 게임이 끝납니다
8. **재시작**: 게임 오버 화면에서 화면을 탭하면 다시 시작할 수 있습니다

## 🛠️ 기술 스택

- **언어**: Kotlin
- **최소 SDK**: Android 7.0 (API 24)
- **타겟 SDK**: Android 14 (API 34)
- **빌드 도구**: Gradle 8.0
- **그래픽**: Android Canvas API
- **프레임워크**: 네이티브 Android (no game engine)

## 🚀 빌드 및 실행 방법

### 사전 요구사항

- Android Studio (최신 버전 권장)
- JDK 8 이상
- Android SDK (API 24 이상)

### 빌드 방법

1. **프로젝트 열기**
   ```bash
   # Android Studio에서 프로젝트 폴더 열기
   File > Open > ddonddo 폴더 선택
   ```

2. **Gradle 동기화**
   - Android Studio가 자동으로 Gradle을 동기화합니다
   - 또는 상단 메뉴에서 `File > Sync Project with Gradle Files` 클릭

3. **빌드 및 실행**
   - 실제 기기를 USB로 연결하거나 에뮬레이터를 실행합니다
   - 상단의 Run 버튼(▶️)을 클릭합니다
   - 또는 `Shift + F10` (Windows/Linux) / `Ctrl + R` (Mac)

### 명령줄에서 빌드

```bash
# 프로젝트 디렉토리로 이동
cd ddonddo

# Gradle wrapper 실행 권한 부여 (macOS/Linux)
chmod +x gradlew

# 디버그 APK 빌드
./gradlew assembleDebug

# 릴리즈 APK 빌드
./gradlew assembleRelease
```

APK 파일 위치: `app/build/outputs/apk/`

## 🤖 자동 빌드 & 릴리스

이 프로젝트는 GitHub Actions를 통해 자동으로 빌드 및 릴리스됩니다!

### 릴리스 방법

1. **버전 업데이트**
   ```kotlin
   // app/build.gradle.kts 파일에서
   versionCode = 3          // 빌드 번호 증가
   versionName = "1.2.0"    // 버전 이름 변경
   ```

2. **변경사항 푸시**
   ```bash
   git add app/build.gradle.kts
   git commit -m "chore: bump version to 1.2.0"
   git push origin main
   ```

3. **자동 실행**
   - GitHub Actions가 자동으로 APK 빌드
   - Release 노트 자동 생성
   - GitHub Release 페이지에 APK 업로드

### 버전 관리
- **Major (X.0.0)**: 주요 기능 추가 또는 호환성 변경
- **Minor (x.X.0)**: 새로운 기능 추가
- **Patch (x.x.X)**: 버그 수정 및 소규모 개선

자세한 내용은 [`.github/workflows/README.md`](.github/workflows/README.md)를 참고하세요.

## 🔊 사운드 파일 설정

게임에 소리 효과를 추가하려면 사운드 파일을 준비해야 합니다.

### 빠른 설정

1. `app/src/main/res/raw/` 폴더에 다음 파일 추가:
   - `jump_sound.mp3` - 점프할 때 소리
   - `game_over_sound.mp3` - 게임 오버 시 소리

2. 무료 사운드 다운로드:
   - [Freesound.org](https://freesound.org/)
   - [Zapsplat](https://www.zapsplat.com/)
   - [Mixkit](https://mixkit.co/free-sound-effects/)
   - [Pixabay](https://pixabay.com/sound-effects/)

**상세 가이드**: `SOUND_SETUP.md` 파일 참고

**참고**: 사운드 파일이 없어도 게임은 정상 작동합니다 (무음으로 실행됨)

## 🎨 게임 커스터마이징

### 난이도 조절

`GameView.kt` 파일에서 다음 값들을 조정할 수 있습니다:

```kotlin
// 장애물 생성 빈도 (프레임 수)
private val obstacleSpawnRate: Int = 90  // 값을 낮추면 더 자주 생성됨
```

`Obstacle.kt`:
```kotlin
// 장애물 속도
private val speed: Float = 12f  // 값을 높이면 더 빠르게 이동
```

`Player.kt`:
```kotlin
// 점프 높이
private val jumpForce: Float = -25f  // 절댓값을 높이면 더 높이 점프

// 중력
private val gravity: Float = 1.5f  // 값을 높이면 더 빠르게 떨어짐
```

### 색상 변경

`app/src/main/res/values/colors.xml` 파일에서 색상을 변경할 수 있습니다.

### 텍스트 변경

`app/src/main/res/values/strings.xml` 파일에서 게임 내 텍스트를 변경할 수 있습니다.

### 🎨 고양이 디자인 특징

게임을 실행하면 귀여운 삼색 고양이가 달리는 모습을 볼 수 있습니다! 🏃‍♂️🐱

- **흰색 몸통**: 깔끔한 흰색 베이스
- **검은 점무늬**: 몸통과 머리, 다리에 검은 점
- **주황색 점무늬**: 몸통과 머리, 다리에 주황색 점
- **검은 왼쪽 귀**: 왼쪽 귀는 검은색
- **주황색 오른쪽 귀**: 오른쪽 귀는 주황색
- **줄무늬 꼬리**: 검은 줄무늬가 있는 꼬리가 흔들림

### 💧 물웅덩이 장애물 디자인

고양이가 싫어하는 물웅덩이가 게임 속에 나타납니다!

- **진한 파란색 베이스**: 물웅덩이의 기본 색상
- **밝은 파란색 레이어**: 깊이감을 주는 중간 레이어
- **하이라이트 효과**: 빛 반사로 반짝이는 물
- **물결 애니메이션**: 주기적으로 반짝이며 살아있는 느낌
- **3가지 크기**: 작은/중간/큰 물웅덩이가 랜덤하게 등장
- **타원형 모양**: 자연스러운 물웅덩이 형태
